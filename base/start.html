设置 Go Proxy ​

输入 go env 查看配置信息

bash
go env

可以使用 go env -w xxx=xxx 修改配置信息

bash
go env -w xxx=xxx

Go Modules ​

使用 go modules 管理依赖

bash
go env -w  GO111MODULE=on  # 默认为 auto
如果我们要使用 gopath 模式，需要 GO111MODULE 设置为 off
bash
go env -w  GO111MODULE=off

尽量不要使用 go path 模式

设置国内镜像源 ​

bash
go env -w GOPROXY=https://goproxy.cn,direct
